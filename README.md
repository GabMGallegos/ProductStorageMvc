# ProductStorageMvc

ASP.NET Core MVC project for storing products with Entity Framework Core and Supabase PostgreSQL.

## Features

- Add product.
- View product list.
- Calculate total value per product.
- Calculate total inventory quantity and total inventory value.
- MVC structure.
- No REST API.
- Dockerfile prepared for Render.
